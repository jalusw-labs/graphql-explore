# GraphQL Server Demo

This is a simple GraphQL server demo using Apollo Server.

## Installation

```bash
npm install
```

## Usage

Start the server:

```bash
npm start
```

Open your browser and navigate to `http://localhost:4000/` to access the GraphQL Playground.

Linting

```bash
npm run lint
```

Testing

```bash
npm test
```

Formatting

```bash
npm run format
```
